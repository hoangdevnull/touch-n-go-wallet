# Touch n Go Wallet

This repository now contains two app implementations:

- `app/`: the original native Android/Kotlin project.
- `touch-n-go-rn/`: the Expo React Native version of the wallet clone.

## React Native App

```bash
cd touch-n-go-rn
npm run start
```

## Android App

Open the repository root in Android Studio and run the Gradle project as before.
